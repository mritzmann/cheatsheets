# Cisco IOS

## Tools

Simulation:

* Cisco Packet Tracer 
* 2911 is a good router for testing

## Basic

Configuration mode:

```
configure terminal
```

Write to disk:

```
write
```

## New Router

Disable ip lookup:

```
no ip domain lookup
```

Set hostname:

```
hostname router01
```

Set password:

```
enable secret <password>
```

## Show

Network configuration:

```
show ip interface brief
```
