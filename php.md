# PHP

## Shell

Run simple php commands:

```shell
php -r 'echo ini_get("memory_limit");'
```

Start dev webserver:

```shell
php -S localhost:8888
```

## Echo

```php
echo "Hello World"
echo "Hello " . "World";
echo("Hello World");
echo "Hello ", "World";
```

## Variables

```php
$greeting = "Hello World";
echo 	$greeting;
```
